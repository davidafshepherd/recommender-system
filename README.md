# Recommender System
- [cosine_similarity_algorithm](cosine_similarity_algorithm) contains the recommender system based on adjusted cosine similarity.
  - Trained on 100K user-item rating pairs ([train set](cosine_similarity_algorithm/test_100k_withratings.csv)).
  - Evaluated on a 100K test set with withheld ratings ([test set](cosine_similarity_algorithm/test_100k_withoutratings.csv)).
- [matrix_factorisation_algorithm](matrix_factorisation_algorithm) contains the recommender system based on matrix factorisation.
  - Trained on 20M user-item rating pairs ([train set](matrix_factorisation_algorithm/test_20M_withratings.csv)).
  - Evaluated on a 20M test set with withheld ratings ([test set](matrix_factorisation_algorithm/test_20M_withoutratings.csv)).
